This project aims to connect MCU UART via USB-to-serial chip, using CH340C. USB D+/D- signals are connected to CH340C through necessary series resistors; since they are differential signals, differential routing is required. For ESD protection, SRV05-4 is added. On the UART side, TX, RX, GND, and 3.3V are connected to MCU, with signals brought out via pin headers. Series resistors of 22-100 ohms are added between TX and RX. Also, to prevent current surge during programming, a 10uF local storage capacitor is placed close to CH340.
