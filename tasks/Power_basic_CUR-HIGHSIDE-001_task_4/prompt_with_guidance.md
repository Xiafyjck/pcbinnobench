This is a high-side current-sense amplifier KiCad project that uses an INA240-class current-sense amplifier to measure the shunt resistor voltage drop on a 24V industrial power bus. Please use the project files, project notes, and `datasheets/` to review whether the INA240 IN+ / IN- inputs have protection against input stresses such as industrial load disconnects and bus transients, and make the necessary schematic changes.

This task specifically checks whether there are existing transient clamp components in the sampling chain R1 (shunt resistor) → R2 / R3 (input-side series placeholders) → INA240 IN+ / IN-, after R2 / R3 and before the INA240 inputs. If this chain has only R2 / R3 as series nodes and no input transient clamp, add an input clamp network after R2 / R3 and before the INA240 inputs. A typical implementation is to place a pair of diodes on each of IN+ and IN-, respectively clamping toward the device positive supply rail and ground, for example using a multi-channel small-signal bidirectional clamp diode array such as BAV199DW in an SOT-363 package, so that overvoltage energy generated by industrial load disconnects / bus transients is clamped between the device supply and ground before reaching the INA240 inputs. Apart from adding the clamp network and connecting it to the existing IN+ / IN- traces, do not modify the INA240 pin connections, gain network, R1 / R2 / R3 values, or other unrelated components and layout.
