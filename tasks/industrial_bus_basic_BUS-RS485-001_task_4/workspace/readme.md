This project implements failsafe idle biasing for RS485 bus. It uses SP3485 as RS485 transceiver, with DE signal controlling both DE and RE# to switch between transmit/receive direction. B line is pulled up to VCC via 10kΩ resistor, A line is pulled down to GND via 10kΩ resistor, implementing external failsafe idle biasing. A 120Ω termination resistor is reserved between A and B, default NC, soldered only when this node is at the physical end of the bus.
