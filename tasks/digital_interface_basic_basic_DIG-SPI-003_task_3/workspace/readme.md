This system uses STM32 MCU to connect two SPI Flash devices W25Q32 and W25Q128 via shared SPI bus (SCK, MOSI, MISO). Each Flash has its own chip select CS pin; device switching is achieved by pulling different CS low in time division. The deselected Flash sets its MISO pin to high impedance to avoid bus signal conflicts. Both Flash support the same SPI modes 0/3 and are powered by 3.3V, so no level shifting is needed. In PCB layout, the SPI bus adopts a star topology centered on the master device, with branches as equal length as possible and close to the corresponding Flash to ensure signal integrity.
